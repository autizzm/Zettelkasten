# –ö–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –¥–µ—Ç–µ—Ä–º–∏–Ω–∞—Ü–∏–∏

–ö–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –¥–µ—Ç–µ—Ä–º–∏–Ω–∞—Ü–∏–∏ $R^2$ —è–≤–ª—è–µ—Ç—Å—è –º–µ—Ä–æ–∏ÃÜ, –ø–æ–∑–≤–æ–ª—è—é—â–µ–∏ÃÜ –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å, –≤ –∫–∞–∫–æ–∏ÃÜ —Å—Ç–µ–ø–µ–Ω–∏ –Ω–∞–∏ÃÜ–¥–µ–Ω–Ω–∞—è –ø—Ä—è–º–∞—è —Ä–µ–≥—Ä–µ—Å—Å–∏–∏ –¥–∞–µ—Ç –ª—É—á—à–∏–∏ÃÜ —Ä–µ–∑—É–ª—å—Ç–∞—Ç –¥–ª—è –æ–±—ä—è—Å–Ω–µ–Ω–∏—è –ø–æ–≤–µ–¥–µ–Ω–∏—è –∑–∞–≤–∏—Å–∏–º–æ–∏ÃÜ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–∏ÃÜ $Y$, —á–µ–º –≥–æ—Ä–∏–∑–æ–Ω—Ç–∞–ª—å–Ω–∞—è –ø—Ä—è–º–∞—è $Y = \bar{y}$.


> –í —Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–µ –∏–Ω–æ–≥–¥–∞  –Ω–∞–∑—ã–≤–∞—é—Ç ¬´–∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç–æ–º –æ–±—ä—è—Å–Ω—ë–Ω–Ω–æ–π –¥–∏—Å–ø–µ—Ä—Å–∏–∏¬ª. –¢.–∫. –æ–Ω –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç –∫–∞–∫–∞—è –¥–æ–ª—è $Y$ –æ–±—ä—è—Å–Ω—è–µ—Ç—Å—è $X$. –°–ª–µ–¥—Å—Ç–≤–µ–Ω–Ω–æ $1- R^2$  ‚Äî –¥–æ–ª—è $Y$, –æ–±—ä—è—Å–Ω—è–µ–º–∞—è $\varepsilon$.


$TSS = RSS + ESS$

–≥–¥–µ
- $TSS = \sum_{t=1}^n (y_t - \bar{y})^2$ - total sum square
- $RSS = \sum_{t=1}^n (\tilde{y_t} - \bar{y})^2$ - residual sum square
- $ESS = \sum_{t=1}^n (y_t - \bar{y})^2$ - error sym square



–ö–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –¥–µ—Ç–µ—Ä–º–∏–Ω–∞—Ü–∏–∏ –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è **$R^2$** –∏ –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è —Ç–∞–∫:

# $R^2 = 1 - \frac{RSS}{TSS} = \frac{ESS}{RSS} = 1 - \frac{\sum_{t=1}^n (\tilde{y_t} - \bar{y})^2}{\sum_{t=1}^n (y_t - \bar{y})^2} =\frac{\sum_{t=1}^{n} (y_i - \tilde{y}_t)^2}{ \sum_{t=1}^{n} (y_t - \bar{y})^2}$

–≥–¥–µ:
- $\tilde{y_t}$‚Äã ‚Äî –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –º–æ–¥–µ–ª–∏,
- $\bar{y}$ ‚Äî —Å—Ä–µ–¥–Ω–µ–µ –Ω–∞–±–ª—é–¥–∞–µ–º–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π $y$.
- $y_t$ ‚Äî —Ñ–∞–∫—Ç–∏—á–µ—Å–∫–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ (–Ω–∞–±–ª—é–¥–µ–Ω–∏–µ), —Ç–æ –Ω–∞ –æ—Å–Ω–æ–≤–µ —á–µ–≥–æ –º—ã —Å—Ç—Ä–æ–∏–ª–∏ –º–æ–¥–µ–ª—å –∏ –ø–æ–ª—É—á–∞–ª–∏ –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã.

**–°–º—ã—Å–ª:** $R^2$ –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç, –∫–∞–∫–∞—è —á–∞—Å—Ç—å **–æ–±—â–µ–π –≤–∞—Ä–∏–∞—Ü–∏–∏ $y$** –æ–±—ä—è—Å–Ω—è–µ—Ç—Å—è –º–æ–¥–µ–ª—å—é.
