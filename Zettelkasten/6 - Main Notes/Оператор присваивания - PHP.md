# Оператор присваивания - PHP

> [!note] 
> В одной строке можно привоить значения нескольким переменным.
> ```php
> <?php
> $n = $m = $p = 3;
> echo "n = ", $n, " m = ", $m, " p = ", $p;
> ?>
> ```

В php есть комбинированные операторы:
- `+=`
- `-=`
- `/=`
