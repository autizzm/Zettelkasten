# F-—Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∞

–û—Ç–ª–∏—á–Ω—ã–π –≤–æ–ø—Ä–æ—Å ‚Äî **F-—Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∞ –∏ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –§–∏—à–µ—Ä–∞** –∑–∞–Ω–∏–º–∞—é—Ç —Ü–µ–Ω—Ç—Ä–∞–ª—å–Ω–æ–µ –º–µ—Å—Ç–æ –≤ –ø—Ä–æ–≤–µ—Ä–∫–µ –∑–Ω–∞—á–∏–º–æ—Å—Ç–∏ —Ä–µ–≥—Ä–µ—Å—Å–∏–æ–Ω–Ω—ã—Ö –º–æ–¥–µ–ª–µ–π.  
–ù–∏–∂–µ –∫—Ä–∞—Ç–∫–æ–µ –∏ —Å—Ç—Ä–æ–≥–æ–µ –æ–±—ä—è—Å–Ω–µ–Ω–∏–µ —Å —Ñ–æ—Ä–º—É–ª–∞–º–∏ –¥–ª—è **–ø—Ä–æ—Å—Ç–æ–π –ª–∏–Ω–µ–π–Ω–æ–π —Ä–µ–≥—Ä–µ—Å—Å–∏–∏ (–ü–õ–†)** –∏ **–º–Ω–æ–∂–µ—Å—Ç–≤–µ–Ω–Ω–æ–π –ª–∏–Ω–µ–π–Ω–æ–π —Ä–µ–≥—Ä–µ—Å—Å–∏–∏ (–ú–õ–†)**.

## üìå 1. –†–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –§–∏—à–µ—Ä–∞

**–†–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –§–∏—à–µ—Ä–∞‚Äì–°–Ω–µ–¥–µ–∫–æ—Ä–∞** ( F_{(k_1, k_2)} ) ‚Äî —ç—Ç–æ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –æ—Ç–Ω–æ—à–µ–Ω–∏—è –¥–≤—É—Ö –Ω–µ–∑–∞–≤–∏—Å–∏–º—ã—Ö —Ö–∏-–∫–≤–∞–¥—Ä–∞—Ç —Å—Ç–∞—Ç–∏—Å—Ç–∏–∫, –¥–µ–ª—ë–Ω–Ω—ã—Ö –Ω–∞ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–µ —Å—Ç–µ–ø–µ–Ω–∏ —Å–≤–æ–±–æ–¥—ã:  

### $F = \frac{ \dfrac{\chi^2_{1}}{k_1} }{ \dfrac{\chi^2_{2}}{k_2} }$
–≥–¥–µ:

- $\chi^2_1 \sim \chi^2(k_1)$
    
- $\chi^2_2 \sim \chi^2(k_2)$
    
- $k_1, k_2$ ‚Äî —Å—Ç–µ–ø–µ–Ω–∏ —Å–≤–æ–±–æ–¥—ã —á–∏—Å–ª–∏—Ç–µ–ª—è –∏ –∑–Ω–∞–º–µ–Ω–∞—Ç–µ–ª—è.
    

üëâ $F$-—Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –≤—Å–µ–≥–¥–∞ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ –∏ –∞—Å–∏–º–º–µ—Ç—Ä–∏—á–Ω–æ.

## üìå 2. –ó–∞—á–µ–º –Ω—É–∂–Ω–∞ F-—Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∞ –≤ —Ä–µ–≥—Ä–µ—Å—Å–∏–∏

$F$-—Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è **–ø—Ä–æ–≤–µ—Ä–∫–∏ –æ–±—â–µ–π –∑–Ω–∞—á–∏–º–æ—Å—Ç–∏** –º–æ–¥–µ–ª–∏:  

### $H_0: \beta_1 = \beta_2 = \dots = \beta_k = 0$

(–≤—Å–µ –æ–±—ä—è—Å–Ω—è—é—â–∏–µ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –Ω–µ –≤–ª–∏—è—é—Ç –Ω–∞ $Y$ )  
–ø—Ä–æ—Ç–∏–≤  

### $H_1: \text{–•–æ—Ç—è –±—ã –æ–¥–∏–Ω } \beta_j \neq 0$


üìå –ï—Å–ª–∏ ( F ) –≤–µ–ª–∏–∫–∞ –∏ –ø—Ä–µ–≤—ã—à–∞–µ—Ç –∫—Ä–∏—Ç–∏—á–µ—Å–∫–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ ( F_{crit} ), —Ç–æ –≥–∏–ø–æ—Ç–µ–∑–∞ ( H_0 ) –æ—Ç–≤–µ—Ä–≥–∞–µ—Ç—Å—è ‚Äî –∑–Ω–∞—á–∏—Ç, –º–æ–¥–µ–ª—å –≤ —Ü–µ–ª–æ–º **—Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∏ –∑–Ω–∞—á–∏–º–∞**.

---

## üìå 3. –î–µ–∫–æ–º–ø–æ–∑–∏—Ü–∏—è –¥–∏—Å–ø–µ—Ä—Å–∏–π

–í –æ—Å–Ω–æ–≤–µ F-–∫—Ä–∏—Ç–µ—Ä–∏—è –ª–µ–∂–∏—Ç —Ä–∞–∑–ª–æ–∂–µ–Ω–∏–µ –æ–±—â–µ–π —Å—É–º–º—ã –∫–≤–∞–¥—Ä–∞—Ç–æ–≤:

 
### $TSS = RSS + ESS$


- $TSS$ ‚Äî total sum of squares (–æ–±—â–∞—è –≤–∞—Ä–∏–∞—Ü–∏—è $y$):  
    
    $TSS = \sum_{i=1}^n (y_i - \bar{y})^2$  
    
- $RSS$ ‚Äî regression sum of squares (–æ–±—ä—è—Å–Ω—ë–Ω–Ω–∞—è –≤–∞—Ä–∏–∞—Ü–∏—è):  
    
    $RSS = \sum_{i=1}^n (\tilde{y}_i - \bar{y})^2$
    
    
- $ESS$ ‚Äî error sum of squares (–Ω–µ–æ–±—ä—è—Å–Ω—ë–Ω–Ω–∞—è –≤–∞—Ä–∏–∞—Ü–∏—è):  
      
    $ESS = \sum_{i=1}^n (y_i - \tilde{y}_i)^2$
    
    

$\tilde{y}_i$ ‚Äî –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –ø–æ –º–æ–¥–µ–ª–∏.

---

## üìå 4. F-—Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∞: –æ–±—â–∞—è —Ñ–æ—Ä–º—É–ª–∞


### $F = \frac{\text{–°—Ä–µ–¥–Ω–∏–π –∫–≤–∞–¥—Ä–∞—Ç —Ä–µ–≥—Ä–µ—Å—Å–∏–∏ (MSR)}}{\text{–°—Ä–µ–¥–Ω–∏–π –∫–≤–∞–¥—Ä–∞—Ç –æ—à–∏–±–∫–∏ (MSE)}}  = \frac{RSS / k}{ESS / (n - k - 1)}$

–≥–¥–µ:

- $k$ ‚Äî —á–∏—Å–ª–æ –æ–±—ä—è—Å–Ω—è—é—â–∏—Ö –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö (–±–µ–∑ —Å–≤–æ–±–æ–¥–Ω–æ–≥–æ —á–ª–µ–Ω–∞),
    
- $n$ ‚Äî –æ–±—ä—ë–º –≤—ã–±–æ—Ä–∫–∏,
    
- —Å—Ç–µ–ø–µ–Ω–∏ —Å–≤–æ–±–æ–¥—ã: —á–∏—Å–ª–∏—Ç–µ–ª—å $k$, –∑–Ω–∞–º–µ–Ω–∞—Ç–µ–ª—å $( n - k - 1 )$.
    

–†–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ:  
  
### $F \sim F_{(k,; n - k - 1)}$

–ø—Ä–∏ —Å–ø—Ä–∞–≤–µ–¥–ª–∏–≤–æ–π $H_0$.

---

## üìå 5. **–ü—Ä–æ—Å—Ç–∞—è –ª–∏–Ω–µ–π–Ω–∞—è —Ä–µ–≥—Ä–µ—Å—Å–∏—è** (–ü–õ–†, ( k=1 )):


### $F = \frac{RSS / 1}{ESS / (n - 2)}= \frac{RSS}{ESS / (n - 2)}$

üìå –í —Å–ª—É—á–∞–µ –ü–õ–† $F$-—Ç–µ—Å—Ç –∏ $t$-—Ç–µ—Å—Ç —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω—ã:  
 
$F = t^2,$ 

–≥–¥–µ $t$ ‚Äî $t$-—Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∞ –¥–ª—è –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç–∞ $\beta_1$.

## üìå 6. **–ú–Ω–æ–∂–µ—Å—Ç–≤–µ–Ω–Ω–∞—è –ª–∏–Ω–µ–π–Ω–∞—è —Ä–µ–≥—Ä–µ—Å—Å–∏—è** (–ú–õ–†, ( k > 1 )):


### $F = \frac{RSS / k}{ESS / (n - k - 1)}= \frac{R^2 / k}{(1 - R^2) / (n - k - 1)}$

(—á–µ—Ä–µ–∑ $R^2$ ‚Äî –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –¥–µ—Ç–µ—Ä–º–∏–Ω–∞—Ü–∏–∏).

–ê–Ω–∞–ª–æ–≥–∏—á–Ω–æ:  
  
### $F = \frac{ (R^2 / k) }{ (1 - R^2) / (n - k - 1) }.$  

–≠—Ç–æ —É–¥–æ–±–Ω–∞—è —Ñ–æ—Ä–º–∞, –µ—Å–ª–∏ $SSR$ –Ω–µ –ø–æ—Å—á–∏—Ç–∞–Ω –Ω–∞–ø—Ä—è–º—É—é.

## üìå 7. –ü—Ä–∞–≤–∏–ª–æ –ø—Ä–∏–Ω—è—Ç–∏—è —Ä–µ—à–µ–Ω–∏—è

    - –µ—Å–ª–∏ $F \le F_{crit}$ ‚Üí –Ω–µ—Ç –æ—Å–Ω–æ–≤–∞–Ω–∏–π –æ—Ç–≤–µ—Ä–≥–Ω—É—Ç—å $H_0$.
        

–≠–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ ‚Äî —Å—Ä–∞–≤–Ω–∏—Ç—å $p$-value —Å $\alpha$.
