# Получение данных из Model - Thymeleaf

Получение данных из Model в html
```html
<!DOCTYPE html>  
<html lang="en" xmlns:th="http://thymeleaf.org">
	<body>
		<p th:text="${message}"/>
	</body>
</html>
```

NOTE: импортировали схему `xmlns:th="http://thymeleaf.org"`
